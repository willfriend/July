How do we put out the fire?

The tools to reduce conflict

As human beings, we all share a common set of psychological mechanisms that can drive intergroup conflict and inhibit conflict resolution.

From polarization to ethnic violence, social media has intensified intergroup conflict because it taps directly into the cognitive and psychological mechanisms that predispose us to conflict.

Behavioral scientists have tested and refined techniques to overcome these mechanisms. At [Organization Name] we leverage social media to scale these techniques and alleviate intergroup conflicts all around the globe.

Content + Conflict Resolution

We create content - video, visual and audio - that incorporates rigorously tested methods for overcoming psychological barriers to conflict resolution.

Awareness

Many psychological mechanisms can be reduced via simple awareness. We create engaging content that educates people about these mechanisms.

Refuting Information

Other mechanisms are overcome by refuting information. We produce content that refutes the implicit beliefs underlying these mechanisms.

Redirection

Still other mechanisms can be attenuated by redirecting the mechanism so that it actually promotes conflict resolution. We make content that reframes salient issues to reverse these mechanisms.
